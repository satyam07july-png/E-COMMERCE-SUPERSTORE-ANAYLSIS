# E-COMMERCE-SUPERSTORE-ANAYLSIS
Developed an end-to-end E-commerce analytics project using Python, SQL, and Power BI. Cleaned and transformed raw data using Python, performed analysis with SQL, and built an interactive 4-page dashboard to visualize sales trends, customer behavior, and profit insights for data-driven decision making.
🚀 E-Commerce Sales Analytics Dashboard
📌 Project Overview

This project is an end-to-end data analytics solution built using Python, SQL, and Power BI. It focuses on analyzing e-commerce sales data to extract meaningful insights related to sales performance, customer behavior, and profitability.

🧰 Tech Stack
Python → Data Cleaning & Preprocessing
SQL (MySQL) → Data Analysis & Querying
Power BI → Data Visualization & Dashboard
📊 Dashboard Features
🟢 Overview
Total Sales, Profit, Orders, Customers
Sales trend over time
Region-wise and category-wise performance
🔵 Sales Analysis
Top selling products
Monthly sales trends
City and category-wise sales
🟣 Customer Analysis
Top customers
Orders per customer
Customer distribution by region
🔴 Profit Analysis
Profit margin insights
Loss-making products
Category and region-wise profit
🧠 Key Insights
Identified high-performing regions and products
Detected loss-making products for business optimization
Analyzed customer purchasing behavior
Evaluated profitability across categories
⚙️ Data Processing
Cleaned raw dataset using Pandas
Handled missing values and duplicates
Created calculated fields (profit, margin, etc.)
Stored and queried structured data using SQL
📈 Power BI Highlights
Interactive 4-page dashboard
KPI cards and dynamic visuals
Filters and slicers for deep analysis
Clean and professional UI design
